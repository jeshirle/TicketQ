In this file you will find the src file for TicketQ. 
Inside this source file you will find two source package files labeled Ticket and TicketQ.
Place this src file into any empty project in any IDE and you will be able to run the code.
The main method is found in the TicketQ.java code.
TicketQ.java is what you will run to access the program. 
TicketQ.java imports from two java files: Ticket and TicketList.
All java files are needed to run the program successfully.
